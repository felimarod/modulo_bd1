## Install

##### with npm
```
npm install -g balzss/cli-typer
```

##### manually
```
git clone https://github.com/GLUD/cli-learn-typing.git
cd cli-learn-typing
npm install
npm build
npm install -g .  # optional
node dist/index.js
```

## Dependecies

- [node.js](https://github.com/nodejs/node) >= 6.4.0
- Typescript
